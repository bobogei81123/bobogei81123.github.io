Hello World to Hexo

2016-07-20

Hexo 是在 StaticGen 上排名第三的 static site generator. 可以幫你快速的建出 blog 等等的靜態網站.

設定方式

以下是在 hexo 3.2.2 和 arch linux 4.6.4 下的設定說明.

首先用 npm 在 global 安裝 hexo.

接著在你想要的地方開一個新的資料夾存放 blog 的檔案. 然後初始化 hexo.

最後在 ./config.yml 裡面更改設定, 找出 Deployment 一區的選項, 這裡假設你要用 Github 當做 deploy 的位置.
[your username] 填入你的 github 帳號.

最後 generate 和 deploy 就會在 [your username].github.io 上看到你的網頁了!
